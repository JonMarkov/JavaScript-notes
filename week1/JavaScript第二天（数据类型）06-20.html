JavaScript新知识

数据类型

布尔类型：true false
Boolean:把其他数据类型转换成布尔值；
    其他数据类型转换成布尔类型是false有且只有五个值：0 "" NaN null undefined;
    逻辑运算符：
    ||：或（只要其中有一个为ture，整体结果是ture）；
    &&：并且（只要有一个是false，整体结果就是false）；
    ！：取反；（首先会把后面的值转成布尔值，然后在取反）；
    ！！：取反再取反（直接把当前值转成布尔值）；
    1，放在if条件中：||：只要其中有一个为true，整体结果是true；&&：只要有一个是false，整体结果就是false；
    2，赋值作用：
    var abc = NaN||undefined (如果前面值转布尔是ture，直接取前面这个值；如果转布尔值false，直接取后面的值)；
    var bcd = null && 17 (如果前面这个值转布尔是true，直接取后面这个值；如果转布尔是false，直接取前面的值)；
null和undefinde：都代表没有；
    null： typeof null -->"object";空对象指针；
    区别：null 表示现在没有，但是以后可能会有；
         undefined：现在没有，以后也不能有；
    常见的null的情况：
        1，通过id获取元素，如果id名字不存在，那么返回null；
        2，通过正则进行捕获时，如果没有捕获到内容，那么结果是null；
        3，在清空对象，销毁堆内存时，给对象赋值时，赋值为null；
    常见的undefined的情况：
        1，当获取对象属性名对应的属性值时如果属性名不存在，那么返回值是undefined；
        2，函数的形参如果没有实参赋值，在函数中默认存储undefinde；
        3，如果函数没有return，那么函数的返回值是undefined；
        4，如果变量只声明，没有赋值，那么默认存储也是undefined；
对象：
    描述：对象由键值对组成，键值对由属性名和属性值组成，键值对用逗号隔开；
    属性名和属性值用冒号隔开；属性名是字符串类型，属性值是数据类型中的一种；
    没有length属性，没有索引；
    对象的定义：
        1，首先会开辟一个新的空间地址；空间地址是16进制；0-9a-f；
        2，把键值对存储到当前这个堆内存下；
        3，把这个空间地址给了对象名；
        对象名实际存储的是一个空间地址；
        在js中遇到[],{}都会开辟新的空间地址；
    对象的增删改查：
        1，新增：
            var obj = {num:1,str:"abc"}
            (1)对象名.属性名 = 属性值  --> obj.a = 100
            (2)对象名["属性名"] = 属性值 --> obj["12"] = 13
            如果属性名是数字的话，会按照数字从小到大顺序显示；
        2，查询：如果属性名不存在，那么获取的属性值是undefined；
            (1)var v = obj.num
            (2)var b = obj["num"]
        3,修改：与新增的区别是属性名如果存在就是修改，没有就是新增；
            (1)对象吗.属性名 = 属性值  --> obj.d = 100
            (2)对象名["属性名"] = 属性值 --> obj["33"] = 13
        4,删除：
            (1)假删除 把键值对中数据定为null -->obj.num = null
            (2)真删除 delete 对象吗.属性名  -->delete obj.num
        5,in:判断当前属性名在对象中是否存在，如果存在返回true，不存在返回false；
            (1)语法：（"属性名" in 对象名）
数组：由数组成员组成，数组之间用逗号隔开 有索引和length
        数组成员都有一个与之对应的索引；从0开始；
        有length属性，代表数组成员的个数
数据类型之间的比较：
        (1)=：赋值
        (2)==: 比较  返回一个布尔值 NaN和自己都不相等，会返回false
        数据类型比较的规律：
        对象==对象：比较的是空间地址
        对象==字符串： 对象默认调用toString方法，先转字符串，然后再进行比较
        对象==布尔：对象先转字符串，再转数字；布尔直接转数字
        对象==数字：对象先调用toString转换成字符串，然后再转number
        布尔==数字：布尔转数字
        字符串==数字：字符串转数字
        布尔==字符串：布尔转数字，字符串也转数字，然后进行比较
        null==undefinde：true
        null和undefinde和其他数据比较都是false
        (3)!:取反，首先会把后面的值转换成布尔值，然后在进行取反
        (4)===：绝对比较：不会进行数据类型之间的转换，只要数据类型不一样，就是false；
        ！：后面跟等号，代表不等于
        ！==：绝对不等于(不会进行类型转换,在比较时除了对值进行比较以外,还比较两边的数据类型,===的非形式)
        ！=：会进行数据类型之间的比较；（在表达式两边的数据类型不一致时,会隐式转换为相同数据类型,然后对值进行比较.）
        ！值：代表的取反
        简单理解就是 != 会转换成相同类型 进行比较,!== 除了比对值还比对类型

操作语句

三个判断语句
    1，if else（if   else if 。。。else）
    （1）if（条件）{
            当括号中条件为true时，执行此处代码；
        }else{
            如果括号中条件不成立，此处代码是不执行的；
        }
    （2）if(条件1){
            条件1为true时执行此处代码
        }else if（条件2）{
            条件1为false，条件2为true时执行此处代码
        }else{
            上述所有条件为false时，执行此处代码
        }
    2,三元运算符：一般用于处理简单的逻辑
        ？：问好前面放的是条件，问好后面放的是当条件成立时执行的代码
        ：：当条件不成立时，执行的代码
        例：条件？如果条件成立则执行此处：条件不成立则执行此处；
            1=="1"?console.log(100):console.log(200)
    3，switch case：拿括号中的值和case后面的值进行的是绝对比较
            break：中止 中断；
            default：默认执行；
            在条件中如果没有break，会继续向下执行，即便条件不成立；直到出现break；
        例：var num = 0;
            switch (num){
                case "1":
                    console.log("1");
                    break;
                case false;
                    console.log("2");
                    break;
                cese 0:
                case 10:
                    console.log("等于0或是10则执行")；
                default:
                    console.log（"结束"）
                
            }
