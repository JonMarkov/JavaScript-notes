数组方法
1、push:
    （1）向数组的末尾新增一项；
    （2）需要传参；
    （3）新数组的数组成员个数；
    （4）原数组发生变化；
2、pop：
    （1）删除数组的最后一项；
    （2）不需要传参数；
    （3）被删除的那一项；
    （4）原有数组发生改变
    （注：ary.length--也是删除最后一项）
3、unshift：
    （1）向数组的开头新增一项
    （2）需要传参，新增的参数
    （3）新数组的数组成员个数；
    （4）愿数组发生变化
4、shift：
    （1）删除数组的第一项；
    （2）不需要传参数；
    （3）被删除的那一项；
    （4）原有数组发生改变；
5、slice：
    （1）数组的截取
    （2）参数为开始的索引m和结束的索引n
    （3）截取后的数组
    （4）原有的数组不发生变化
     用法：ary.slice(m,n) --> 
     截取数组中索引m项到索引n项，包括索引m项，不包括索引n项；
     如果传一个参数m，则是从索引m截取到末尾；
     如果不传参数，则复制整个数组；
     如果传的参数索引为负数的话，让当前的length加上当前负数为索引值；
     如果索引n小于索引m，则返回空数组；
6、splice：
    （1）删除数组中的某几项
    （2）参数为从最开始的索引m开始往后删除n个（包括m项）
    （3）删除的数组
    （4）原有数组发生变化
    用法：删除数组中从索引m开始往后的n个，包括m项
    ary.splice(m,n):从索引m开始，删除n个
    ary,splice(m):从索引m开始，删除到末尾
    ary.splice(0):删除数组
    ary.splice(m,x,n)替换从索引m开始，删除x个，用n替换（n可是多个）
7、sort:
    （1）数组排序（只能排序）
    （2）
        sort（）：只能数组成员是相同位数的数组；
        sort（function（a，b）{return a-b}）数组从小到大排序
        sort（function （a,b）{return b-a}）数组从大到小排序
    （3）排序后的数组
    （4）原有数组发生改变
8、reverse:
    （1）将数组颠倒过来
    （2）不需要传参数
    （3）数组成员倒序之后的数组
    （4）原数组发生改变
9、concat：
    （1）数组的拼接
    （2）
        不传参：数组的克隆
        传参：（数组或是每一项）；把传入的实参拼接新的数组中
    （3）拼接之后的新数组；
    （4）原有数组不发生改变
    结构：var ary = [1,2,3]
        var ary1 = [2,3,4]
        ary.concat(ary1)//ary数组和ary1拼接
        ary.concat（100，200）//数组ary增加两个值100和200
10、join：
    （1）把数组成员按照特定的字符连接成一个字符串
    （2）
        不传参数：默认按照逗号分开
        传参数：按照特定的字符（传的参数）
    （3）拼接之后的字符串
    （4）原有数组不发生改变
11、indexOf
    （1）检测数组成员在数组中从左到右第一次出现的索引位置；判断当前项是否在数组中存在，如果不存在，则返回-1
    （2）需要传被检测的参数
    （3）返回在数组中从左到右第一次出现的索引
    （4）原有数组不发生改变
12、lastIndexOf：
    （1）检测数组成员在数组中从右到左第一次出现的索引位置；判断当前项是否在数组中存在，如果不存在，则返回-1
    （2）需要传被检测的参数
    （3）返回在数组中从右到左第一次出现的索引
    （4）原有数组不发生改变
13、map：
    结构：var ary = [1,2,3,4,5,2,3,4]
            ary.map(function (item,index){

    （4）原有数组不发生改变

冒泡排序

数组的冒泡排序

实现从小到大排序；
原理：让相邻两项进行比较，如果前面一项比后面一项大，那么让这两项互换位置；如果前面比后面一项小，那么不换位置；每循环一轮，把数组的最大值放到数组的最后面；
结构：
var

数组的插入排序

实现从小到大排序
原理：去原有数组中去取每一项，然后和新数组中的成员进行比较，从新数组后面开始，一一向前比较，比新数组项小，继续和前一项比较，直到当前项比数组中一项值大时，把它放到这一项的后面；
结构：
var  newAry = [];
newAry.push(ary[0]);
for(var i=1;i<ary.length;i++){
    var cur = ary[i];
    for(var j=newAry.length-1;j>=0;j--){
        if(cur<newAry[j]){// 如果数组中拿到的这一项比新数组中j的这一项小；那么继续j--;和上一个成员进行比较；
            if(j===0){
                newAry.unshift(cur);
                break;// 只会中止当前的小循环
            }
        }else{
            // cur<newAry[j]的情况，把当前项放到新数组这一项的后面；
            newAry.splice(j+1,0,cur);
            break;
        }
    }

}

// 从前向后
console.log(newAry);

    get；获取；
    element：元素
    by：通过；
    ID：id名字；
    id是唯一的，不能重复；
2、document.getElementsByTagName（通过标签名来获取元素）
    不强制用document来获取；
    得到的是一个类数组集合；
3、document.getElementsByClassName（通过class获取元素）
    不强制用document来获取；
    得到的是一个类数组集合；
    兼容性：IE8及以下是不兼容的；
4、document.getElementsByName（通过name属性来获取元素）
    不强制用document来获取；
    兼容性：在IE9及以下，会默认把ID声明的获取到；
5、document.documentElement（获取当前的文件--html）
